## 🚀 启动步骤

### 1. 安装依赖（如果还未安装）

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问：http://localhost:5173/

### 3. 配置代理

已在 `vite.config.ts` 中配置代理：
- `/api` → `http://localhost:8080`
- `/oauth2` → `http://localhost:8080`

确保后端服务运行在 8080 端口。

## 💡 使用说明

### 登录测试

默认登录参数：
- 用户名：`admin`
- 密码：`admin123`

OAuth2 客户端配置：
- client_id: `jnet-pkce-client`
- client_secret: `jnet-secret`
- grant_type: `password`

### API 调用示例

```typescript
import { userApi } from '@/api'

// 查询用户列表
const res = await userApi.pageUser({ pageNum: 1, pageSize: 10 })

// 获取用户详情
const user = await userApi.getUserById(1)

// 新增用户
await userApi.addUser(userData)

// 删除用户
await userApi.deleteUser(userId)
```

### 权限判断

```typescript
import { useUserStore } from '@/stores/user'

const userStore = useUserStore()

// 检查权限
if (userStore.hasPermission('system:user:list')) {
  // 显示按钮或菜单
}
```
